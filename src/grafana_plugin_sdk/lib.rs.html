1/*! The Grafana Plugin SDK for Rust.
2
3This crate contains a Rust implementation of the Grafana plugin SDK. It is divided into three main modules:
4
5- [`backend`] contains the traits that must be implemented by backend plugins for various pieces of functionality,
6  whether querying data, calling resources, or streaming data between Grafana and the plugin.
7- [`data`] contains the fundamental data structures used by backend plugins, such as [`Frame`][data::Frame]s, [`Field`][data::Field]s,
8  and their associated metadata.
9- [`live`] contains functionality used by [Grafana Live], the streaming messaging service available from
10  Grafana 8.0.
11
12The [`prelude`] contains some useful unambiguous traits which are helpful when creating some structures,
13particularly [`Frame`][data::Frame]s and [`Field`][data::Field]s.
14
15Backend plugins communicate with Grafana via gRPC. The low-level protocols are exposed in the [`pluginv2`]
16module as an escape hatch, if required. Please file an issue if this is needed and we will try to
17accommodate your needs in the next release of the high-level SDK.
18
19See the docs on [backend plugins on grafana.com] for an introduction to backend Grafana plugins, or check out
20the [crate examples] or [sample app repo] to get started with writing a backend plugin in Rust.
21
22# Feature flags
23
24The following feature flags enable additional functionality for this crate:
25
26- `reqwest` - adds an [`IntoHttpResponse`][crate::backend::IntoHttpResponse] implementation for
27  [`reqwest::Response`]
28
29[Backend plugins on grafana.com]: https://grafana.com/docs/grafana/latest/developers/plugins/backend/
30[Grafana Live]: https://grafana.com/docs/grafana/latest/live/
31[crate examples]: https://github.com/grafana/grafana-plugin-sdk-rust/tree/main/examples
32[sample app repo]: https://github.com/sd2k/grafana-sample-backend-plugin-rust/
33*/

77#[doc(hidden)]
78pub fn async_main<R>(fut: impl std::future::Future<Output = R> + Send) -> R {
79    tokio::runtime::Builder::new_multi_thread()
80        .thread_name("grafana-plugin-worker-thread")
81        .enable_all()
82        .build()
83        .expect("create tokio runtime")
84        .block_on(fut)
85}
